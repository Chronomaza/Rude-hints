# Rude-hints
Ever found spore's hints annoying? Ever felt frustrated by unclear "this creation cannot be saved" messages? Do you like being called mean words while you play videogames? Then boy do I have the mod for you!
This is my first mod for spore and it could possibly be a bit rough around the edges in some spots, but in my testing works just fine.
Not compatible with better save warnings, as this mod changes every error message and 'hint' popup that's not downright unused and technically serves the same purpose because of that.
